# sumododda.me

Personal portfolio website built with [Next.js](https://nextjs.org/) and deployed on [Cloudflare Pages](https://pages.cloudflare.com/).

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Static output is generated in the `out/` directory.

## Cloudflare Pages

| Setting              | Value           |
| -------------------- | --------------- |
| Build command        | `npm run build` |
| Build output directory | `out`         |

## Customization

Edit `data/sitedata.js` to update your personal info, projects, skills, and social links.

## Tech Stack

- **Framework** — Next.js 15 (App Router, static export)
- **Styling** — Sass (SCSS)
- **Hosting** — Cloudflare Pages
